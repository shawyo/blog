H5竖屏下横屏显示

2017-10-22
笔记

有一天经理说：哎~那个谁，咱这个页面能不能做成，竖屏横着显示，横屏也横这显示的…

想一想大概的实现思路是这样的

  1. 选取一个 dom 节点比如 body ;
  2. css 赋值宽等于实际的高，高等于实际的宽;
  3. 利用 css3 transform rotate 属性选转90度;
  4. js 监听 onorientationchange 旋转控制样式;
